Feladat

Orvosi betegnyilvántartás Készítsünk relációs adatmodellt egy háziorvosi betegnyilvántartó rendszerhez. Az orvosok elvárásainak megfelelően az adatbázisnak tartalmaznia kell a betegek személyi adatait, gyógyszerérzékenységüket, az egyes vizsgálatok időpontjait és felírt gyógyszereket.

    Feltételezzük, hogy:
  • egy vizsgálaton több betegség is diagnosztizálható
  • egy betegségre több gyógyszer is felírható
  • egy páciens több gyógyszerre is lehet érzékeny
  • egy vizsgálatot a dátum és a vizsgálat kódja határoz meg egyértelműen

Adatbázis kódja

CREATE DATABASE korhaz;
USE korhaz;

CREATE TABLE paciens (
pid INT PRIMARY KEY,
nev VARCHAR(100) NOT NULL,
cim VARCHAR(200),
telefon VARCHAR(20)
);

CREATE TABLE gyogyszer (
gynev VARCHAR(100) PRIMARY KEY
);

CREATE TABLE betegseg (
bnev VARCHAR(100) PRIMARY KEY
);

CREATE TABLE vizsgalat (
vid INT PRIMARY KEY,
pid INT,
datum DATE,
FOREIGN KEY (pid) REFERENCES paciens(pid)
);

CREATE TABLE mire_erzekeny (
pid INT,
gynev VARCHAR(100),
PRIMARY KEY (pid, gynev),
FOREIGN KEY (pid) REFERENCES paciens(pid),
FOREIGN KEY (gynev) REFERENCES gyogyszer(gynev)
);

CREATE TABLE diagnozis (
vid INT,
bnev VARCHAR(100),
datum DATE,
PRIMARY KEY (vid, bnev),
FOREIGN KEY (vid) REFERENCES vizsgalat(vid),
FOREIGN KEY (bnev) REFERENCES betegseg(bnev)
);

CREATE TABLE mit_kap (
gynev VARCHAR(100),
bnev VARCHAR(100),
PRIMARY KEY (gynev, bnev),
FOREIGN KEY (gynev) REFERENCES gyogyszer(gynev),
FOREIGN KEY (bnev) REFERENCES betegseg(bnev)
);

Lekérdezések

  1. Melyik páciens milyen betegséget kapott diagnózisként

    SELECT paciens.nev, vizsgalat.vid, diagnozis.datum, diagnozis.bnev FROM paciens JOIN vizsgalat ON paciens.pid = vizsgalat.pid JOIN diagnozis ON vizsgalat.vid = diagnozis.vid ORDER BY paciens.nev;

  2. Melyik betegségre milyen gyógyszert adnak

    SELECT betegseg.bnev, gyogyszer.gynev FROM betegseg JOIN mit_kap ON betegseg.bnev = mit_kap.bnev JOIN gyogyszer ON mit_kap.gynev = gyogyszer.gynev ORDER BY betegseg.bnev;

  3. Mely páciensek érzékenyek valamelyik gyógyszerre

    SELECT paciens.nev, gyogyszer.gynev FROM paciens JOIN mire_erzekeny ON paciens.pid = mire_erzekeny.pid JOIN gyogyszer ON mire_erzekeny.gynev = gyogyszer.gynev ORDER BY paciens.nev;

  4. Páciens betegsége és az arra adott gyógyszer

    SELECT paciens.nev, betegseg.bnev, gyogyszer.gynev FROM paciens JOIN vizsgalat ON paciens.pid = vizsgalat.pid JOIN diagnozis ON vizsgalat.vid = diagnozis.vid JOIN betegseg ON diagnozis.bnev = betegseg.bnev JOIN mit_kap ON betegseg.bnev = mit_kap.bnev JOIN gyogyszer ON mit_kap.gynev = gyogyszer.gynev ORDER BY paciens.nev;

  5. Hány páciens kapott el egy adott betegséget

    SELECT betegseg.bnev, COUNT(paciens.pid) AS paciensek_szama FROM betegseg JOIN diagnozis ON betegseg.bnev = diagnozis.bnev JOIN vizsgalat ON diagnozis.vid = vizsgalat.vid JOIN paciens ON vizsgalat.pid = paciens.pid GROUP BY betegseg.bnev;
